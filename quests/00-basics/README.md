# Quest 00: Absolute Basics

Start here if the student knows nothing about programming.

## Concepts

- A constant names a value.
- Strings are text.
- Numbers can be added/subtracted.
- Booleans are `true` or `false`.
- A function takes input and returns output.

## Exercise

Run:

```sh
bun run quest:00
```

Read `exercise.ts` and `exercise.test.ts` side by side.

## Break/Fix Drill

1. Run `bun run quest:00` and see it pass.
2. Change `startingGems` from `100` to `50`.
3. Run `bun run quest:00` again and read the failure.
4. Change it back.
5. Change `double` from `value * 2` to `value * 3`.
6. Run it again.

The goal is to learn that tiny code changes produce exact test changes.
