# Quest 01: JavaScript Basics

Teach only the things needed to read ordinary TypeScript files.

## Concepts

- `const` and `let`
- Function parameters
- Return values
- Arrays
- Objects
- `map`, `filter`, `find`

## Exercise

Open `exercise.ts`, explain each function out loud, then run:

```sh
bun run quest:01
```

Change one function at a time and watch the test fail/pass.

## Break/Fix Drill

1. Run `bun run quest:01` and see it pass.
2. Change `unit.name` to `unit.id` in `getUnitNames`.
3. Run `bun run quest:01` and read the failure.
4. Change it back.
5. Run the command again.

Do not continue until the student can explain what changed.
