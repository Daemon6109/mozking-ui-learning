# Quest 01: JavaScript Basics

Teach only the things needed to read ordinary TypeScript files.

## Concepts

- `const` and `let`
- Function parameters
- Return values
- Arrays
- Objects
- `map`, `filter`, `find`

## Exercise

Open `exercise.ts`, explain each function out loud, then run:

```sh
bun run quest:01
```

Change one function at a time and watch the test fail/pass.
